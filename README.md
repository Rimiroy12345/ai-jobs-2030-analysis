# AI Impact on Jobs — 2030

**Project:** AI Impact on Jobs by 2030  
**Authors:** Pournami Roy, Vrukshita Vinod, Tanishka Tiwari, Suryawanshi Prachiti Prashant, Palak Soni

## Summary
This project analyzes a dataset of 3,000 job records to investigate automation risk, AI exposure, and key skills required by 2030. Includes EDA, visualizations, and recommendations.

## Files
- `Python_miniproject_ppt.pdf` — project slides / summary
- `notebooks/` — Jupyter notebooks with EDA and plots
- `data/AI_Impact_on_Jobs_2030.csv` — dataset 
- `README.md` — this file
